# Demo Remote Media Enterprise

Progetto di collaudo AI-HTML 1.4.0 con:

- logo principale e footer via URL;
- header topbar-nav;
- menu rich con immagini via URL;
- footer CTA con background remoto;
- hero SBS con video e poster remoti;
- nessun media obbligatorio nella Media Library.

## Ordine import

1. Importare `theme-options.json` da `AI-HTML → Opzioni JSON`.
2. Importare `menus.json` da `AI-HTML → Menu JSON` con sostituzione attiva.
3. Creare una pagina Home con template `smart-site-home.php`.
4. Importare `sbs-home.json` nel builder della pagina.
5. Impostare la pagina come homepage statica.

## Risultati attesi

- Logo grafico remoto nella navbar.
- Topbar con telefono, email e link progetto.
- Navbar sticky solida.
- Mega menu Servizi con immagini remote.
- Hero video autoplay muted con poster remoto.
- Footer CTA con due bottoni.
- Logo grafico remoto nel footer.
- Sfondo footer remoto con overlay.
- Tre trust item personalizzati.

## Nota

Gli URL di terze parti sono solo fixture dimostrative. In produzione sostituirli con un CDN controllato dal proprietario del sito.
